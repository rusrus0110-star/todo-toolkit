# Personal Task Manager

A personal task manager built with React, Vite, Redux Toolkit, Ant Design, and dnd-kit.

## Features

- Create, edit, and delete tasks
- Add task descriptions
- Manage task status: To do / In progress / Done
- Set priority: Low / Medium / High
- Set due dates
- Organize tasks by projects
- Create a new project directly from the task form
- Filter tasks by status, priority, and project
- Search tasks by title and description
- Sort tasks by date or priority
- Reorder tasks with drag & drop in manual mode
- Persist data in localStorage

## Tech Stack

- React
- Vite
- Redux Toolkit
- React Redux
- Ant Design
- dnd-kit
- dayjs

## Installation

```bash
npm install
```
